Educational Quiz • 牙齒小百科

清潔牙齒

親愛的小朋友，準備好要跟著牙刷超人一起學習如何保護牙齒了嗎？讓我們開始挑戰吧！

01

小朋友，每天早上起床和晚上睡覺前，我們都要做什麼事來保護牙齒呢？

A 玩玩具
B 刷刷牙
C 看電視
正確答案：(B)
02

刷牙的時候，我們需要用到哪一對好朋友？

A 牙刷和牙膏
B 筷子和湯匙
C 梳子和鏡子
正確答案：(A)
03

擠牙膏的時候，要擠多少才是最合適的呢？

A 擠得像小山一樣高
B 只要像一顆小豌豆那麼大
C 把整條牙膏都擠完
正確答案：(B)
04

牙刷在牙齒上跳舞時，要把哪裡刷乾淨呢？

A 只刷門牙
B 只有外面要刷
C 裡面、外面和咬東西的地方都要刷
正確答案：(C)
05

刷牙的時候，如果嘴巴裡有白白的泡泡，應該怎麼辦？

A 吞下去，甜甜的很好吃
B 吐在洗手槽裡，再用清水漱口
C 噴在鏡子上玩
正確答案：(B)
06

下面哪一種食物是牙齒最喜歡的健康好朋友？

A 甜甜圈和糖果
B 蘋果和青菜
C 汽水和可樂
正確答案：(B)
07

誰是專門幫我們檢查牙齒、趕走蛀牙蟲蟲的超人？

A 警察叔叔
B 牙醫官
C 消防員
正確答案：(B)
08

牙刷用了很久，毛毛都翹起來像開花一樣，該怎麼辦？

A 繼續用，沒關係
B 請爸爸媽媽幫忙換一支新的牙刷
C 拿剪刀把毛剪掉
正確答案：(B)
09

吃完甜甜的蛋糕或餅乾後，如果不能馬上刷牙，應該先做什麼？

A 喝水漱漱口
B 趕快再吃一顆糖
C 直接去睡覺
正確答案：(A)
10

躲在牙齒縫縫裡的「蛀牙蟲蟲」最怕什麼？

A 牙刷和牙線
B 躲貓貓
C 睡覺
正確答案：(A)
11

刷牙的時間要多久，才能把牙齒刷得亮晶晶？

A 隨便刷一秒鐘
B 唱完兩遍「刷牙歌」的時間（約2-3分鐘）
C 刷一整個下午
正確答案：(B)
12

我們應該多久給牙醫官檢查一次牙齒呢？

A 牙齒痛的時候才去
B 每半年（六個月）去檢查一次
C 一輩子去一次就好了
正確答案：(B)
13

牙線是做什麼用的呢？

A 用來綁頭髮
B 清潔牙刷刷不到的牙齒縫縫
C 用來翻書
正確答案：(B)
14

想要牙齒長得又壯又健康，可以多喝哪種飲料？

A 牛奶
B 紅茶
C 咖啡
正確答案：(A)
15

刷牙的時候，舌頭也要輕輕刷一下嗎？

A 要，因為舌頭也會藏細菌
B 不要，舌頭不用洗澡
C 舌頭會自己變乾淨
正確答案：(A)
16

為什麼我們不能吃太多的糖果？

A 因為糖果太硬了
B 因為蛀牙蟲蟲最喜歡吃糖，會弄壞我們的牙齒
C 因為糖果不好吃
正確答案：(B)
17

下面哪一個動作是正確的刷牙姿勢？

A 很大力地橫著亂刷
B 輕輕地繞圈圈或是上下刷
C 用力敲牙齒
正確答案：(B)
18

牙刷是全家人共用的，還是每個人都要有自己的？

A 爸爸媽媽和我的都放在一起共用
B 每個人都要有自己專屬的牙刷
C 誰先拿到就誰用
正確答案：(B)
19

刷完牙之後，牙齒摸起來應該是什麼感覺？

A 黏黏膩膩的
B 滑滑乾淨的
C 苦苦的
正確答案：(B)
20

擁有健康的牙齒，笑起來會怎麼樣？

A 會變得很醜
B 會變得很可愛，而且很有自信
C 會沒辦法說話
正確答案：(B)

挑戰完成！👏

恭喜小朋友完成了所有的題目！記得要天天乖乖刷牙，讓牙齒亮晶晶喔！
